HOW DOES THE INTERNET WORKS

Lot of people think of the internet as something hanging in the cloud, super complex and difficult to understand. NO!

It's just a long piece of wire that connects different computers to each other to communicate and transfer data.

Some of this computers have a special job: to be online 24/7 ready to serve you all of the data and files that you are requesting when you try accessing websites and they are called SERVERS

Any computer that you use to access the internet is called a CLIENT

A web server is like a giant library opened 24/7 that you can go in at any time of the day and request any web page or information and it will be able to serve you with all of the files and data that you will need to be able to view whatever website it is that you requested.

How do you quickly locate the things you want in this library?

When you search for a homepage on your computer (client) the browser sends a message to your Internet Service Provider (ISP), the ISP then sends it to a DNS (Domain Name System) server will search it's database for the exact ip address of the website you are trying to access. And once the dns server finds that address, it sends it back to your browser.

And now you can send a direct request to that address through your isp. And this message will be delivered via what's called the internet backbone. which is literally the backbone of internet.

Internet backbone is a very high-speed data transmission line that provides networking facilities to relatively small but high-speed Internet service providers all around the world. The largest and fastest networks are linked together with fiber-optic connections and high-performance routers.

the dns server is basically like a phonebook that matches domain names with their unique ip addresses.

Every computer that is connected to the internet has an ip address which is a like a unique postal address for your computer for sending and receiving files.

(To find the direct address) client → ISP → DNS → ISP → client

(Direct request to the address) client → ISP → internet backbone → Server → internet backbone → ISP → client

HOW DO WEBSITES WORK?

In order to access a web page you need a web browser. these are softwares that allow you look up the ip address of your website and to be able to receive data that it then renders it into the website you see. The data that the browser receives usually consists of 3 types of files. HTML, CSS and JAVASCRIPT.

The 3 building blocks of the web

  1. html : responsible for the structure of your website. Imagine your website as a house, html would be the builder who would come and build the walls.

    e.g. Image, button, textbox etc

  1. css : responsible for styling your website. Used to modify the appearance

    The painters and decorators adding stylistic flourishes to your house making it look the way you want it to.

    eg. the button you got from your html, you want the color to be green, you want the font to be a certain size. All these are done using css code.

  1. javascript : adds functionality to your site. Allows it do things and have behaviour rather than just display images and text to us. Like the electrician that comes in and connects the wires so that the light bulb switches on or somebody who comes in and does the plumbing so that the toilet can actually flush. And javascript does that.
